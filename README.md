### 2. (예시) LLM 기반 RAG 챗봇 서비스

- **설명**: 특정 도메인 문서를 기반으로 답변하는 RAG 기반 챗봇
- **기술 스택**: Python, FastAPI, LangChain, FAISS/ChromaDB, OpenAI API(or Ollama)
- **주요 기능**
  - PDF/문서 임베딩 및 벡터 검색
  - 사용자 질문 기반 문서 검색 + 답변 생성
  - 간단한 웹 UI 연동 (Streamlit/React 등)
- **역할**: 설계 · 백엔드 · 모델 연결 전반
